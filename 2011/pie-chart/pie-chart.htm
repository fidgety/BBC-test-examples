An approach to displaying an accessible pie chart

My approach is to initially display the data in a datatable - the most accessible format. I have then used a JavaScript object to expose the data and properties of the table, and this in turn is consumed by a simple object wrapping some Raphael calls. The pie chart overlays the datatable in this example, but given time I would add the option to show either using tabs.

Although the code is written to be reusable it would need a custom pie chart generator which would gracefully handle different dimensions and quantities of values.

Ideally the SVG would be created server side as part of the content management, but this example assumes this isn't feasible.
